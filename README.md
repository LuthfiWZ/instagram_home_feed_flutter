## Deskripsi Singkat
Membuat tampilan Instagram Home Feed sederhana dengan:
- stories
- 3 postingan
- icon interaksi

## Widget yang Digunakan
- Scaffold → struktur halaman
- AppBar → header aplikasi
- ListView → menampilkan stories dan post
- CircleAvatar → foto profil
- Image.network → gambar postingan
- Row & Column → layout
- RichText → caption postingan
